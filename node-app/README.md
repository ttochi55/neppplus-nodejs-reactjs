# NodeJS


## Express

Install

```bash
$ npm install express
```

Run Server

```bash
$ node index.js
```

## Express Generator

Install

```bash
$ npx express-generator --view=ejs server
```

Run Server

```bash
$ cd server
$ npm install
$ npm start
```
